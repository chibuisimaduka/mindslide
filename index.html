What is MindSlide?

  • Mindslide is a Mindmap viewer. It turns your Freeplane / Freemind mindmaps into a stunning HTML presentation in a few seconds.
  • Demo mindmap
  • Demo HTML presentation
  • Freeplane software

Features

  • MindSlide currently supports

  • speaker notes (from the node notes) hit "s"

  • 2D flow + overview mode. hit "escape" or "o"    easy navigation (space, arrows, click)

  • simple & rich-text nodes, including code highlight 

    @page:right { 
  margin: 1cm 2cm 1.3cm 4cm; 
}

  • links on nodes. ctrl+k in Freeplane

  • images and nodes connectors ↝

  • gestures, markdown contents, PDF export, JavaScript API and more.

  • works in most recent browsers

Not supported

  • Many elements are still not supported. We welcome any pointers how to best do it, as well as help to do so!
  • node and edge style
  • node icons
  • clouds
  • "post-it" nodes are not marked differently from other nodes
  • node attributes
  • images zoom
  • author information
  • summary nodes
  • Known Bugs ↴
  • Put your suggestions on our "issues" list!

Known Bugs

  • Various things we should fix, any pointer how to do so is welcome.

  • improve styles, fonts and general support of richtext content, as for now some elements don't show or not nicely

  • nodes with only an image show as list item anyway.
  • rich text is not fully shown as simple text in indexes & titles ↝
  • Report your bugs on our "issues" list!

Requirements

  • You need a recent browser with javascript support (Chrome and Firefox recommended)
  • browsers supported by reveal.js

Install MindSlide

  • To install MindSlide takes about 2 minutes:

  • Install from the archives:

    1. dowload and unpack https://github.com/berteh/mindslide/archive/master.zip into "mindslide" directory (or pick any name you like)
    2. download and upack https://github.com/hakimel/reveal.js/archive/master.zip into mindslide/reveal.js directory
  • copy mindslide/mindslide.xsl into your freeplane/resources/xslt directory
  • launch freemind and select file > export > MindeSlide Presentation (.html) to export your mindmap

  • save the HTML presentation in mindslide directory! (for now)

Configure

  • Many options are available
  • edit directly the mindslide.xsl (or the exported .html presentation file) for your prefered theme, transition, extra behaviour or functionality.
  • see reveal.js configuration
