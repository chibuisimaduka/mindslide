What is MindSlide?

  • Mindslide is a Mindmap viewer. It turns your Freeplane / Freemind mindmaps into a stunning HTML presentation in a few seconds. This very presentation has been generated from our documentation mindmap.
  • Demo mindmap
  • Demo HTML presentation
  • Freeplane software

Features

  • MindSlide currently supports
  • presenter notes (from the node notes), hit "s" to get the presenter view
  • 2D slides presentation and overview mode (hit 'escape' or 'o')
  • easy navigation (space, arrows, click)

  • simple & rich-text nodes, including code highlight 

    @import url('print.css'); 
@page:right { 
  margin: 1cm 2cm 1.3cm 4cm; 
}
  • clickable links (ctrl+K in freeplane)
  • images
  • nodes connectors ↝

  • gestures and more, including including nested slides, markdown contents, PDF export, speaker notes and a JavaScript API

  • decent browser support and fall-back
  • Not supported ↴
  • Bugs ↴

Not supported

  • Many elements are still not supported. We welcome any pointers how to best do it, as well as help to do so!
  • node and edge style
  • node icons
  • clouds
  • "post-it" nodes are not marked differently from other nodes
  • node attributes
  • images zoom
  • author information
  • summary nodes
  • Put your suggestions on our "issues" list!

Bugs

  • Various things we should fix, any pointer how to do so is welcome.
  • encode &gt; < in text <nodes>
  • improve styles, fonts and general support of richtext content, as for now some elements don't show.
  • nodes with only an image show as list item anyway.
  • Report your bugs on our "issues" list!

Requirements

  • We recommend to use Firefox. Most other recent browsers are supported but have not been tested or require additional software: ↝
  • You need a recent browser with javascript support (firefox recommended)
  • browsers supported by reveal.js
  • browsers supported by jquery

Install xsl export variant

  • To install MindSlide takes about 2 minutes:

  • Install from the archives:

    1. dowload and unpack https://github.com/berteh/mindslide/archive/master.zip into "mindslide" directory (or pick any name you like)
    2. download and upack https://github.com/hakimel/reveal.js/archive/master.zip into mindslide/reveal.js directory
  • copy mindslide/mindslide.xslt into your freeplane/resources/xslt directory
  • launch freemind and select file > export > HTML document (html;html5) to export your HTML presentation

  • save the export in mindslide directory! (for now)

Install javascript variant (not recommended)

  • To install MindSlide takes about 3 minutes, you have 2 options:

  • Install from the archives:

    1. dowload and unpack https://github.com/berteh/mindslide/archive/master.zip into "mindslide" directory (or pick any name you like)
    2. download and upack https://github.com/hakimel/reveal.js/archive/master.zip into mindslide/reveal.js directory

  • Or install from source code repository:

    • git clone git@github.com:berteh/mindslide.git
    • dir mindslide   ("cd mindslide" in linux/ubuntu/mac)
    • git submodule update --init
  • copy your (freemind or freeplane) mindmap file (something.mm) as mm.xml file into mindslide directory
  • copy its related images (if any) in their respective directory accessible from your webserver
  • browse to mindslide.html page and enjoy.

Other browsers

  • MindSlide works works out-of-the box in Firefox. Other browsers will be added to this list when reported to work.

  • Chrome is not able to load the data of your mindmap from a local file, you therefore need to run a webserver (any will do) and put mindslide in a web directory.

  • if you have no webserver we recommend Mongoose: download it in your mindslide directory, execute and browse to http://localhost:8080/mindslide.html with your favorite browser.
